# WingSuiteApp
Application that runs the WingSuite web app

## Requirements
* `Node.js` (v18.0.0 or greater)
* `npm` (v8.0.0 or greater)

## Setup
1. Install all npm packages: `npm install`
2. Run application: `npm run dev`
3. Go to website: `http://localhost:3000/`
